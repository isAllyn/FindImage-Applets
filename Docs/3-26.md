### 创建项目

`vue create -p dcloudio/uni-preset-vue my-project`

### 启动项目

1.  `npm run dev:mp-weixin`

> 可进行更改

2. 将打包后的 dist->mp-weixin 导入到微信开发工具

### eslint 规则配置

> https://blog.csdn.net/weixin_55333190/article/details/123722082

### Plugin

`npm i -D sass`

### 安装 uni ui

`npm i @dcloudio/uni-ui`
安装后一定要按需导入
